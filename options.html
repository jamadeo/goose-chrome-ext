Open in Goose — Settings

Configure URL patterns. When a page matches, a button appears in the lower-right that opens the Goose desktop app with your prompt.

Pattern & prompt syntax

Pattern

  • * matches one path segment (no /)
  • ** matches anything, including /
  • The scheme is optional — https:// is assumed if missing
  • Query strings and hashes are ignored when matching

Placeholders inside your prompt

  • <URL> — the full canonical page URL (no query/hash)
  • <1>, <2>, … — the values matched by each * / ** in order

Example

  • Pattern: github.com/*/*/pull/*
  • Prompt: Review the PR at <URL> in repo <1>/<2>.
